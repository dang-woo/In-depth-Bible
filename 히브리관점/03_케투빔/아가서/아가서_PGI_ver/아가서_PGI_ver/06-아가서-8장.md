# 아가서 8장: 영원한 연합과 부활

8장은 아가서의 결론이자 1:1–2:7(서론)과 **카이즘 구조**로 대응하는 결미이다. 서론에서 제기된 모든 주제가 여기서 성취되며, 신부의 부활(휴거)과 영원한 연합이 선포된다. 1장에서 "나의 포도원을 지키지 못하였다"던 신부가 "내 포도원은 내 앞에 있다"고 고백하는 완성의 장이다.

---

## 8:1–2 오빠처럼 — 공적 친밀함의 갈망

> "네가 나의 어미 젖을 먹은 형제 같았더라면 내가 밖에서 만나도 입을 맞추어도 사람들이 나를 업신여기지 아니하였을 것을"

- 오빠(형제) = 공개적 관계 → 연인 관계는 숨겨야 하지만 형제 관계는 드러낼 수 있음
- 신부의 소원: 주님과의 친밀함을 일상 속에서 **당당하게** 살아내고 싶다
- "내 어미 집으로 이끌어 가겠다" = 교회로 임재를 가져옴 (3:4와 동일)
- "석류즙을 섞은 포도주" = 토라(석류 613)와 은혜(포도주)의 혼합 → 완전한 양식

## 8:3 왼손과 오른손

> "그의 왼손은 내 머리 아래 있고 그의 오른손은 나를 안는구나"

- **왼손(스몰)** = 숨겨진 섭리, 보이지 않는 인도
- **오른손(야민)** = 능력, 구원, 드러난 역사 (시110:1)
- 2:6과 동일한 표현이지만 맥락이 다름:
  - 2:6 = 소원·갈망 단계
  - **8:3 = 성취·안식 단계**

## 8:4 세 번째 "깨우지 말라" — 노루와 들사슴의 소멸

> "예루살렘 딸들아, 내가 너희에게 부탁한다. 사랑이 원하기 전에 흔들지 말고 깨우지 말라"

### 세 번의 비교

| 위치 | 노루·들사슴 | 신부의 상태 |
|------|--------------|-----------|
| 아2:7 | **있음** | 임재를 배우는 중 |
| 아3:5 | **있음** | 찾아서 만남 |
| **아8:4** | **없음 (사라짐)** | **신부 자체가 부활** |

- 찌비옷(노루) = 쩨바옷(만군), 아얄롯(들사슴) = 엘로힘(하나님 권능)
- 노루와 들사슴이 사라진 것 = 천사의 보호가 더 이상 필요 없음
- **신부가 부활체가 됨** → 천사가 섬기던 자가 이제 천사보다 높아짐
- 이것이 **휴거/부활**의 순간을 가리킴

## 8:5 광야에서 올라오는 자

> "광야에서 올라오며 그 사랑하는 자에게 기대어 오는 자가 누구인가?"

- 3:6과 대응하지만 차이가 있음:
  - 3:6 = 솔로몬의 가마(왕의 행렬)
  - **8:5 = 신부가 직접 올라옴** (더 이상 가마 안이 아님)
- "그 사랑하는 자에게 기대어" = 완전한 의지, 자기 힘이 아닌 그분의 능력
- **광야에서 올라옴** = 출애굽의 완성, 가나안 입성

> "사과나무 아래서 내가 너를 깨웠다. 거기서 네 어미가 네게 해산의 고통을 겪었다"

- **사과나무** = 2:3의 사과나무(신랑)와 동일 → 처음 만난 곳에서의 회복
- "네 어미가 해산의 고통" = 교회(어미)의 산고 속에서 신부가 태어남
- 계12장 여인의 해산 = 남자 아이(이기는 자)의 탄생과 직접 연결

## 8:6–7 여호와의 불꽃 — 사랑의 본질 선언

> "나를 네 마음에 인같이 두고 네 팔에 도장 같이 두라. 사랑은 죽음같이 강하고 질투는 스올같이 잔인하며, 불같이 일어나니 **여호와의 불꽃**이로다"

### 핵심 단어: 살헤베트야(שַׁלְהֶבֶתְיָה)

- 살헤베트(불꽃) + **야(יָה = 하나님 이름)** = "여호와의 불꽃"
- 아가서 전체에서 하나님의 이름이 **딱 한 번** 나타나는 곳
- 하나님의 이름이 "사랑의 불꽃" 안에 숨겨져 있음 → 사랑의 본질이 하나님 자체

| 요소 | 의미 |
|------|------|
| 인(호탐) | 왕의 도장, 정체성의 확인 (학2:23) |
| 죽음같이 강함 | 사랑은 죽음도 이김 |
| 질투(킨아)는 스올같이 | 하나님의 거룩한 질투, 우상을 용납 못함 |
| 여호와의 불꽃 | 꺼지지 않는 불, 히12:29 "하나님은 소멸하는 불" |

> "많은 물도 이 사랑을 끌 수 없고 강물도 잠기지 못하나니, 사람이 자기 집의 모든 재산을 다 주고 사랑과 바꾸려 한다면 멸시를 받으리라"

- **많은 물** = 환난, 시험, 세상의 모든 것 (사43:2)
- 사랑은 **구매 불가** — 은혜(하나님의 일방적 사랑)를 돈으로 사는 것은 모독
- 이것은 아가서의 **신학적 중심 선언**: 하나님의 사랑의 본질

## 8:8–10 작은 누이 — 이스라엘과 열방

> "우리에게 작은 누이가 있는데 아직 유방이 없으니, 그가 청혼을 받는 날에 우리가 어떻게 하리요?"

### 화자: 시온의 딸들 (중동의 유대인·무슬림)

| 요소 | 의미 |
|------|------|
| 작은 누이 | 아직 준비되지 않은 이스라엘 / 열방의 연약한 교회 |
| 유방 없음 | 양육 능력(구약·신약 균형)이 아직 없음 |
| 성벽이면 | 은망대를 세워줌 (지켜줌) |
| 문이면 | 백향목 판자로 두름 (보호·강화) |

## 8:10 신부의 최종 정체성 선언

> "나는 성벽이요 나의 유방은 망대 같으니 그의 눈에 **화평(샬롬)을 얻은 자** 같게 되었구나"

- **성벽** = 흔들리지 않는 정체성 (1:5 "검을지라도"에서 시작한 여정의 완성)
- **유방이 망대** = 양육 능력의 완성, 다른 이를 먹일 수 있음
- **화평을 얻은 자(모쩨엣 샬롬)** = 슐라밋(שׁוּלַמִּית)의 의미 완성
  - 솔로몬(쉴로모) = 대가를 지불한 자(신랑)
  - 슐라밋 = 대가가 지불된 자(신부) = 화평을 얻은 자

## 8:11–12 바알하몬의 포도원 vs 나의 포도원

> "솔로몬에게 바알하몬에 포도원이 있어 지키는 자들에게 맡겼으니, 그 열매를 위하여 각각 은 천을 바치리라"

- **바알하몬(בַּעַל הָמוֹן)** = "군중의 주인" → 적그리스도가 지배하는 세계 시스템 / 제3성전
- "지키는 자들에게 맡김" = 위탁 경영 (일꾼에게 맡겨진 포도원 = 마21:33 포도원 비유)
- **은 천** = 세상의 보상 체계

> "나의 포도원은 내 앞에 있구나. 솔로몬, 천은 네게 돌리고 열매 지키는 자에게 이백을 돌릴지니라"

- "나의 포도원은 내 앞에 있다" = 1:6 "나의 포도원은 지키지 못하였다"의 **완전한 역전**
- 바알하몬의 거대한 포도원보다 **내 앞의 작은 포도원**(주님과의 관계)이 참된 것
- **이백(200)** = 열매 지키는 자의 몫 → 사역자에 대한 합당한 대우

## 8:13 동산에 거하는 자에게

> "동산에 거하는 자야, 벗들이 네 음성을 기다리나니 나에게 들려다오"

- 신랑의 마지막 요청: "네 음성을 들려다오" = 기도·예배·증거
- "벗들이 기다린다" = 천사들과 성도들이 신부의 음성을 기다림
- 이것은 1:2 "내게 입맞추기를 원하니"에 대한 **최종 응답 요청**

## 8:14 신부의 마지막 응답 — 마라나타

> "나의 사랑하는 자야, 빨리 오라. 향기로운 산들 위의 노루와 어린 사슴 같이 하라"

- **"빨리 오라(브라흐)"** = 도망치라 / 서두르라 → 계22:20 "아멘 주 예수여, 오시옵소서"
- **향기로운 산들(하레이 브사밈)** = 향품 산 → 부활·승천의 산
- 이것이 아가서의 마지막 한 마디이자 성경 전체의 마지막 기도: **마라나타**
- 신부의 여정 완성: 갈망(1:2) → 순교(5장) → 부활(8:4) → "빨리 오라"(8:14)

## 1장과 8장의 카이즘 완성

| 1:1–2:7 (서론) | 8장 (결론) |
|----------------|-----------|
| 포도원을 지키지 못함 (1:6) | 내 포도원은 내 앞에 (8:12) |
| 검을지라도 (1:5) | 성벽이요, 화평을 얻은 자 (8:10) |
| 사랑이 포도주보다 나음 (1:2) | 사랑은 죽음같이 강함, 여호와의 불꽃 (8:6) |
| 노루와 들사슴 있음 (2:7) | 노루와 들사슴 사라짐 (8:4) |
| 왕의 침소 초청 (1:4) | 마라나타 — 빨리 오라 (8:14) |
| 사랑의 갈망 시작 | 사랑의 완성과 영원한 연합 |

## 이 구간의 핵심 메시지

1. **살헤베트야**: 아가서에서 유일하게 하나님 이름이 등장하는 곳 — 사랑의 불꽃 안에 하나님이 계심
2. **노루와 들사슴 소멸**: 신부의 부활/휴거 — 천사의 보호가 불필요해짐
3. **포도원의 역전**: 잃어버린 것이 완전히 회복됨
4. **마라나타**: 아가서의 마지막 말 = 성경의 마지막 기도
