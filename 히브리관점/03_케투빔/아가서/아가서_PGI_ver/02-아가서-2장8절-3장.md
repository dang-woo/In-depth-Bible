# 아가서 2장 8절 – 3장: 부활의 초청

2:8–3:11은 신랑(메시아)이 부활하여 신부를 부르시는 장면이다. 겨울(십자가·무덤)이 지나고 봄(부활)이 왔으니 일어나라는 초청, 그리고 신부가 밤에 사랑하는 자를 찾아 나서는 여정이 담겨 있다.

---

## 2:8–9 신랑의 등장 — 산을 넘고 언덕 위에 뛰는 자

> "나의 사랑하는 자의 목소리로다. 보라 그가 산에서 뛰며 언덕을 넘어오는구나"

- **산을 넘는 자** = 부활하여 모든 장벽을 뛰어넘으신 메시아
- **노루(찌비)와 어린 사슴** = 군사적 민첩함, 전사-신랑
- "우리 벽 뒤에 서서 창으로 엿보며 격자 사이로 들여다 본다"
  - 벽 = 죄의 담 / 율법의 담 → 그러나 그분이 먼저 다가오심
  - **격자(하라킴)**: 그물처럼 보이지만 빛이 들어오는 틈 → 말씀이 비추는 순간

## 2:10–13 "일어나 함께 가자" — 부활의 봄 선언

> "겨울도 지나고 비도 그쳤고 지면에 꽃이 피고 새가 노래하는 때가 왔다"

| 이미지 | 의미 |
|--------|------|
| 겨울 끝남 | 십자가·무덤의 기간 종료 |
| 꽃이 핌 | 부활의 첫 열매 |
| 새의 노래(자밀) | 가지치기 때, 비둘기 목소리 |
| 무화과 익음 | 이스라엘의 회복 (무화과 = 이스라엘) |
| 포도꽃 향기 | 포도원(교회)의 부흥 |

- "비둘기(요나) 소리가 들린다" = 성령의 탄식, 기도의 소리
- 무화과의 시작은 [[마24#32]] "무화과나무의 비유를 배워라"와 직접 연결

## 2:14 바위 틈의 비둘기

> "바위 틈에, 절벽의 은밀한 곳에 있는 나의 비둘기야"

- **바위 틈(셀라)** = 십자가에 달린 메시아의 찔린 옆구리, 모세가 숨었던 반석의 틈 ([[출33#22]])
- "네 얼굴을 보이라, 네 음성을 들려다오" = 주님이 신부의 기도를 갈망하심
- 아름다움(나아웨)과 달콤한 목소리(아레브) = 기도가 하나님께는 사랑의 음성

## 2:15 여우를 잡으라

> "우리를 위하여 여우를 잡으라, 포도원을 허는 작은 여우를"

- **작은 여우** = 포도원(친밀함)을 훼방하는 작은 죄, 작은 습관, 작은 타협
- 큰 죄보다 위험한 것이 작은 여우 — 매일의 경건이 무너지는 것
- 포도원이 꽃이 필 때(부흥기)에 오히려 여우가 활발함

## 2:16–17 소속 고백 1단계

> "나의 사랑하는 자는 내게 속하였고 나는 그에게 속하였도다"

- 첫 번째 소속 고백: 아직 **내가** 먼저 → 자기 중심적 단계
- "베데르(갈라진) 산 위에서 노루와 같이 돌아오라" = [[창15장]] 아브라함 언약, 쪼갠 짐승 사이를 지나가는 언약의 횃불

## 3:1–4 밤에 찾는 신부 — 첫 번째 실패와 발견

> "밤에 내 침상에서 내 마음에 사랑하는 자를 찾았으나 찾지 못하였도다"

### 찾는 과정

| 단계 | 내용 | 의미 |
|------|------|------|
| 1 | 침상에서 찾음 | 편안한 곳(comfort zone)에서의 기도 |
| 2 | 일어나 성중에 다님 | 교회(종교 시스템) 안에서 찾음 |
| 3 | 순라군(파수꾼)에게 물음 | 목사/지도자에게 도움 요청 |
| 4 | 지나자마자 발견 | 사람을 지나쳐야 주님을 만남 |

- "만나 놓고 놓지 아니하고 어미 집으로 데려감" = 주님의 임재를 교회(어미 집)로 가져옴
- **어미 집** = 교회, 영적 어머니, 신앙 공동체

## 3:5 두 번째 "깨우지 말라"

> "노루와 들사슴을 두고 부탁한다. 사랑이 원하기 전에 깨우지 말라"

- 두 번째 반복 — 찾아낸 임재를 혼이 침범하지 못하게 하라
- 아직 노루와 들사슴이 **있음** → 신부가 아직 완성되지 않음

## 3:6–11 광야에서 올라오는 솔로몬의 탈것

> "향 연기 기둥 같이 광야에서 올라오는 자가 누구인가?"

### 솔로몬의 가마(아피르욘)

| 요소 | 의미 |
|------|------|
| 레바논 나무로 만듦 | 향백나무(ereZ) = 메시아 |
| 은기둥 | 구속(은 = 속전) |
| 금받침 | 신성(금 = 하나님의 본질) |
| 자색 방석 | 왕의 색, 하늘과 땅의 연합 |
| 사랑으로 수놓음 | 예루살렘 딸들의 사랑 (교회의 기도) |

- **광야에서 올라옴** = 출애굽의 구름기둥·불기둥의 재현
- "솔로몬의 어머니가 씌워준 면류관" = 혼인 날의 기쁨
- 이것은 **혼인** 행렬이자 동시에 **전쟁** 행렬 — 60 용사가 둘러섬 ([[삿7장]] 기드온 300 용사와 연결)
- 밤의 두려움(공포) 때문에 칼을 찬 용사들 = 마지막 때 영적 전쟁의 실체

## 이 구간의 핵심 메시지

1. **부활의 초청**: 겨울이 지나고 봄이 왔으니 일어나라 — 십자가 이후 새 시대가 열림
2. **작은 여우 경계**: 부흥기에 작은 타협이 가장 위험
3. **찾는 자의 여정**: 편안한 자리 → 시스템 → 사람을 지나쳐야 주님을 만남
4. **혼인과 전쟁의 동시성**: 신부의 행렬은 곧 전사의 행렬
